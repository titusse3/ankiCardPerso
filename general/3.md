## Log1

Pour chaque point donner une égaliter :
- \\(log_b(xy)\\)         
- \\(log_b(x/y)\\)
- \\(log_b(x^y)\\)
- \\(log_b(1)\\)
- \\(log_b(b)\\)
- \\(log_b(b^n)\\)
- \\(log_b(1/n)\\)
- \\(-log_b(n)\\)

%

- \\(log_b(xy) = log_b x + log_b y\\)
- \\(log_b(x/y) = log_b x - log_b y\\)
- \\(log_b(x^y) = y * log_b x\\)
- \\(log_b(1) = 0\\)
- \\(log_b(b) = 1\\)
- \\(log_b(b^n) = n\\)
- \\(log_b(1/n) = -log_b n\\)
- \\(-log_b n = log_b(1/n)\\)
